# Mockathon

## Description

Create a todo list app

## Goals

- Define the stack

- Define responsibilites

- Assess how we work as a team

- Sync local environments

- Build app

## Features

- Create todo
- Edit todo
- See todos(Todo table)
- Delete todo(check off todo and add to recycle bin)
- Recycle Bin

## Stack

### Frontend

- React
- JS
- CSS
- Bootstrap

### Backend

- NestJS
- TS
- Mongoose(NestJS Wrapper)

### Database MongoDB
